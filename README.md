BizPulse Growth Dashboard

A modern SaaS-style dashboard built for small businesses and local retail stores to simplify daily operations such as inventory tracking, sales management, customer records, invoicing, GST reporting, profit analytics, and AI-based sales forecasting.

Features

📦 Inventory Management

💰 Sales Tracking

👥 Customer Management

🧾 Invoice Generation

📑 GST Report

📊 Profit Analytics

📱 WhatsApp Notifications

🤖 AI Sales Prediction

📈 Business Growth Insights

Tech Stack

Frontend: React + Vite + Tailwind CSS

Backend: Node.js + Express.js

Database: MongoDB Atlas

Authentication: JWT + bcrypt

AI Module: Python + scikit-learn
